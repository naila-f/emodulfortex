Welcome to E-Modul Materi Format Teks Halaman Web

Website ini dirancang khusus untuk mendukung siswa, terutama mereka yang berada di jurusan Rekayasa Perangkat Lunak (RPL), agar dapat memahami secara mendalam materi tentang format teks pada halaman web. Website ini dilengkapi dengan materi dan video yang disajikan dengan baik. Tidak hanya menyediakan materi dan video, website ini juga memastikan bahwa siswa dapat mengukur pemahamannya melalui evaluasi dan latihan praktikum. Evaluasi akan membantu siswa mengukur sejauh mana pengetahuannya telah berkembang, sedangkan latihan praktikum akan memberi siswa kesempatan untuk menerapkan konsep langsung dalam script.

Fitur Website

MATERI

Berisikan kumpulan sub materi format teks halaman web, mulai dari dasar-dasar struktur teks menggunakan HTMLhingga memformat teks pada halaman web akan dibahas semua konsep dengan jelas dan terperinci. Materi disajikan dengan cara yang mudah dipahami, disertai dengan contoh script dan ilustrasi visual sehingga dapat membantu siswa dalam memahami materi

Materi

LATIHAN PRAKTIKUM

Berisikan halaman yang dapat digunakan siswa memiliki kesempatan untuk secara langsung terlibat dalam praktikum dengan membuat script sendiri. Website ini telah menyediakan lingkungan yang aman dan terstruktur dimana siswa dapat menjelajahi konsep format teks di halaman web dan melihat hasilnya secara langsung.

Praktikum

EVALUASI

Berisikan soal yang terdiri dari pilihan ganda. Setiap soal dirancang untuk menguji pemahaman siswa terhadap materi yang telah dipelajari. Ini adalah kesempatan untuk mengukur sejauh mana siswa telah memahami konsep-konsep penting tentang format teks di halaman web dan sebagai bahan evaluasi.
